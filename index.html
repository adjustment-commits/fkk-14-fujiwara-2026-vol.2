GYM A｜支える・乗る

前脚股関節で受ける土台と、軸脚で乗り続ける感覚づくり。

所要目安 20〜30分 強度 中 テーマ 土台

1. スプリットスタンス・アイソホールド

回数：20〜30秒 × 3セット / 左右

やり方：スプリット姿勢で前脚7〜8割荷重。軽く前傾し、前脚の付け根に体重を乗せたまま保持する。

ポイント：膝で耐えすぎない。股関節で支えている感覚を残す。

2. 前足高スプリットスクワット

回数：6〜8回 × 3セット / 左右

やり方：前脚を5〜10cm高くしてスプリットスクワット。3秒で下がり、下で1秒止めて戻る。

ポイント：反動禁止。下で潰れず、前脚の付け根に体重が残ること。

3. 片脚RDL

回数：5〜6回 × 3セット / 左右

やり方：片脚立ちから股関節で前に倒れ、バランスを保ったまま戻る。

ポイント：足先ではなく股関節で立つ。骨盤が開きすぎない。

4. ヒップイン（バンド）

回数：6回 × 3セット / 左右

やり方：骨盤にバンドを巻き、片脚支持に近い姿勢で軽く内側へ動かす。

ポイント：膝だけ入れない。股関節の奥が締まる感覚を優先する。

5. ハーフニーリング・プレス

回数：6〜8回 × 3セット / 左右

やり方：片膝立ちで前に押す。体幹をまっすぐ保ったまま、反りすぎずに行う。

ポイント：押す時に腰が先に逃げない。下半身が安定した中で上半身を使う。

GYM B｜集める・止める

脚だけで処理せず、受けた力を中心へまとめて静かに止める。

所要目安 20〜30分 強度 中 テーマ 受けと停止

1. ブルガリアンスクワット（スロー）

回数：5〜6回 × 3セット / 左右

やり方：ゆっくり下がって最下点で2秒止める。前脚で体を支えたまま戻る。

ポイント：脚で押し返すより、股関節から体を戻すイメージ。

2. スプリットポゴ → ストップ

回数：3＋1 × 4セット / 左右

やり方：スプリット姿勢で小さく3回弾み、4回目でピタッと止まる。

ポイント：高く跳ばない。止まった時にブレず、静かな接地になること。

3. ラテラルバウンド → ストップ

回数：4〜5回 × 3セット / 左右

やり方：横に小さく跳び、反対脚で着地して2秒静止する。

ポイント：着地後に膝が流れない。骨盤が外へ逃げない。

4. ボックス・ステップオフ → ストップ

回数：4〜5回 × 3セット / 左右

やり方：低い台から降りて着地し、その場で静止する。

ポイント：骨盤の高さが急に落ちない。着地音を静かにする。

5. デッドバグ＋バンド

回数：6回 × 3セット / 左右

やり方：仰向けでバンドを引きながら対側の手足を伸ばす。

ポイント：腰が反らない。脚が動いても体幹が崩れない。

GYM C｜つなげる・出す

下半身で作ったものを体幹から上へ通す。大きくではなく、短く速く。

所要目安 20〜25分 強度 中〜やや高 テーマ 伝達

1. ハーフニーリング・ローテーションプレス

回数：6〜8回 × 3セット / 左右

やり方：片膝立ちで軽く回りながら前へ押す。腰だけを切らず、下から上へ流す。

ポイント：骨盤から胸、そのあと腕の順で動く。

2. MBローテーションスラム

回数：5回 × 4セット / 左右

やり方：軽量メディシンボールで短く回り、素早く床へ叩く。

ポイント：腕だけで叩かない。大きく振り回さず、一気にまとめる。

3. MBチェストパス to ウォール

回数：5回 × 4セット

やり方：壁に向かって短く速くチェストパス。大きく反らない。

ポイント：体幹からまっすぐ出す。肩だけで押し込まない。

4. ステップアップ・ニーアップホールド

回数：5回 × 3セット / 左右

やり方：台に乗り、上で片脚を上げて1秒止める。勢いで通過しない。

ポイント：支える脚の股関節が潰れない。上でまとまる。

5. ケーブル・アンチローテーションホールド

回数：15〜20秒 × 3セット / 左右

やり方：横から引かれる負荷に対して正面を保つ。

ポイント：骨盤だけ、肩だけで耐えず、全身で静かに保つ。

GYMは A → B → C の順で使っても、単独で使ってもよい。1回で全部やる必要はなく、その日のテーマで1パターン選ぶ形でOK。

FIELD A｜止まる

移動から受けへの切り替えを作る。止まる形を揃える日。

所要目安 15〜25分 強度 中 テーマ 停止

1. ウォーク → スプリットストップ

回数：5回 × 3セット / 左右

やり方：数歩歩いて、合図でスプリット姿勢に入り止まる。

ポイント：前脚の付け根で止める。毎回同じ形に入る。

2. ミニハードル → ストップ

回数：4回 × 3セット

やり方：低いハードルを越えて、最後に止まる。

ポイント：急いで流れない。最後の形をそろえる。

3. サイドステップ → ストップ

回数：5回 × 3セット / 左右

やり方：横へ移動して、合図で止まる。

ポイント：膝だけでブレーキをかけず、股関節に体重を残す。

4. MB落下キャッチ

回数：5回 × 3セット

やり方：軽いメディシンボールを受け取り、その場で止まる。

ポイント：上半身だけで受けず、全身で静かに吸収する。

FIELD B｜乗る

片脚支持、移動、バランス。軸脚と着地側のまとまりを作る日。

所要目安 15〜25分 強度 中 テーマ 片脚支持

1. 片脚スキップ → ストップ

回数：10〜15m × 4本 / 左右

やり方：片脚中心でスキップし、最後の接地で2秒静止する。

ポイント：最後にブレずに止まる。片脚で立てる。

2. 片脚ホップ → スティック

回数：4回 × 3セット / 左右

やり方：片脚で小さく前へ跳び、着地で静止する。

ポイント：高く・遠くより、着地の安定を優先する。

3. ウォーキングランジ

回数：10歩 × 2〜3本

やり方：ゆっくり前に進むランジ。毎歩で股関節に体重を乗せる。

ポイント：膝だけに乗らない。歩幅は大きすぎない。

4. ラテラルシャッフル → スプリットキャッチ

回数：4回 × 3セット / 左右

やり方：横移動して最後にスプリット姿勢で止まる。

ポイント：流れたまま終わらず、最後の受けを揃える。

FIELD C｜まとめる

反発、リズム、静止を合わせて、現場で使いやすい形にする日。

所要目安 15〜25分 強度 中〜やや高 テーマ リズムと停止

1. 連続バウンディング → ストップ

回数：10〜15m × 4本

やり方：小さめの連続バウンディングを行い、最後の1歩で静止する。

ポイント：大きく飛びすぎず、最後にまとめることを優先。

2. リズムスキップ

回数：15〜20m × 4本

やり方：一定テンポでスキップ。速さよりリズムをそろえる。

ポイント：接地音と高さをそろえる。雑に急がない。

3. ミニハードル・リズムラン

回数：3〜4台 × 4セット

やり方：低いハードルを一定テンポで越える。

ポイント：速さより気持ちよくそろうこと。最後まで同じリズム。

4. MB受け → 1歩 → ストップ

回数：4〜5回 × 3セット

やり方：軽いMBを受けて1歩だけ進み、最後に止まる。

ポイント：受けて終わらず、最後の1歩まで静かにまとめる。

FIELDは 止める日、乗る日、まとめる日 で分けて使う。グラウンド状況に合わせて1パターンだけ選べばよい。
